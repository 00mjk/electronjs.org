Get Started with Electron

Getting started with Electron is easy. Download a prebuilt binary or use npm to install from the command line. Full documentaion in the repository.

# Install globally in your $PATH
npm install atom-shell -g

# Install as a development dependency
npm install atom-shell --save-dev
