"Userland" is a term borrowed from the Unix community, and recently repopularized by users of Node and npm. The term refers to the vast ecosystem of tools that exist outside of a product's "core". In essence, Node is the core and npm is userland.

Like Node, Electron is focused on having a small set of APIs that provide all the necessary primitives for developing multi-platform Desktop applications. This design philosophy allows Electron to remain a flexible tool without being overly prescriptive about how it should be used. Userland enables users to create and share tools that provide additional functionality on top of what is available in Electron "core".
